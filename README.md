# 用户行为风险分析系统

## 项目简介
本项目基于用户访问日志进行行为分析与风险识别，通过统计用户请求频率、活跃时段、操作分布和访问路径等特征，
结合规则检测方法，对潜在异常用户进行识别，为系统风控、运营分析和安全监控提供支持。

## 技术栈
- Python
- Pandas
- NumPy
- SQL
- Java

## 核心功能
1. 日志数据清洗与加载
2. 用户行为特征提取
3. 风险规则检测
4. 风险报告生成
5. 基础服务模块封装

## 项目亮点
- 从原始日志出发完成数据清洗、特征提取、风险判断、报告输出完整流程
- 使用“规则 + 数据分析”方式完成异常行为识别
- 同时包含 Python 数据分析模块与 Java 服务模块，具备一定工程完整性
